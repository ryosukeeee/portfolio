Re:Act

ドアノブ,Seeduino xiao,サーボモーター、ロータリーエンコーダー


この作品は人の気配や存在感を記録・再生することを目標に制作した作品です。
鑑賞者がドアノブを回転させると、数秒後に同じような回し方で向こうから回し返されます。 ドアノブを回す動作、一種の気配を入力すると、同じように、木霊のように気配を返してくれます。
これによってドアの向こうに何者かの存在感を感じさせる作品となっています。
